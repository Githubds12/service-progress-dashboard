## 🚀 Features
- **Real-time Updates**: Automatically refreshes every 3 seconds with the latest data.
- **Premium UI**: Modern dark-glass aesthetics with smooth animations and charts.
- **Financial Tracking**: Tracks total earnings, daily averages, and projected monthly totals.
- **Strategic Targets**: Recommends daily service counts to stay on track for your goal.
- **Live Indicator**: Pulsating "LIVE" status when the server is active.

## 🛠️ How to Run
1. **Update Data**: `python update_dashboard.py`
2. **Start Server**: `python serve_dashboard.py`
3. **Access**: `http://localhost:8000/Dashboard.html`

## 📊 Tech Stack
- **Frontend**: Vanilla HTML5, CSS3 (Glassmorphism), JavaScript.
- **Backend**: Python (HTTP Server & Data Processing).
- **Charts**: Chart.js.

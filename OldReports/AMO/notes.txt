AMO Analysis Notes:
- API Domain: api.chatie.love
- Auth Flow:
  1. GET /api/polaris/login-reg-sms-v2 -> Sends SMS.
  2. POST /api/polaris/overseas-cellphone-login -> Login with OTP.
- Security:
  - 'sign' parameter is required in all requests.
  - Testing shows 'sign' remains valid for the same phone number for extended periods (likely doesn't include a strict timestamp).
  - Custom headers required: App-Key: 111001, Platform: android, Version-Name: 2.30.0.
- Automation: High feasibility as the 'sign' generation can likely be replicated or even hardcoded for specific numbers during testing.
